# Butane config builder for Core OS's (Fedora Core OS, Alt Container OS etc.)

1. Create your server config `srv.bu` from `srv.bu.example`
1. Run `docker compose up`
2. Get your builded Ignition file `srv.ign`.

> See [Butane](https://coreos.github.io/butane/), [Ignition](https://coreos.github.io/ignition/)
